# Saswat Saidutta — Legal Portfolio Website

Personal portfolio website for Saswat Saidutta, a BBA LLB student and emerging legal professional.

**Live site:** https://saswat-legal.vercel.app *(update after deployment)*

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks, no build step)
- Google Fonts: Cormorant Garamond + Inter
- Deployed on Vercel

## Local Development

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/saswat-legal.git
cd saswat-legal

# Open in browser (no build needed)
open index.html

# Or use a local server
npx serve .
```

## Project Structure

```
saswat-legal/
├── index.html      # Main HTML
├── style.css       # All styles
├── script.js       # Scroll animations & nav
└── README.md
```

## Deployment (Vercel)

This site is deployed via Vercel with zero configuration.

1. Push to GitHub
2. Import repo at vercel.com
3. Click Deploy — done.
